# LKS MILITAR - TACTICAL RADAR HORIZON & LOS SIMULATOR

Technical analysis of electromagnetic wave propagation, spatial line-of-sight (LOS), and radar horizon constraints.

## Overview

This repository contains a standalone, high-fidelity mathematical simulator developed by the LKS MILITAR project. 

The tool calculates and visualizes the degradation of the line-of-sight between emitting platforms (radar systems, electro-optical sensors) and target vectors. It bypasses generic geometric models by implementing atmospheric refraction corrections based on terrestrial tropospheric density.

All mathematical models are based on official documentation, scientific literature, and standard geodetic parameters.

## Scope

- Electromagnetic Wave Propagation
- Atmospheric Refraction Modeling
- Radar Horizon and Shadow Zones
- Line-of-Sight (LOS) Geometric Analysis
- Aerospace Defense Systems
- Airborne Early Warning and Control (AEW&C) constraints

## Technical Specifications

The simulation engine supports the following atmospheric correction factors:

- **ITU-R Standard (4/3 Earth Radius):** k=0.25 applied to microwave propagation (X, L, and S band radars).
- **MDPI / Geodetic Standard:** k=0.14 for the visible optical spectrum, FLIR, and Laser designation systems.
- **Strict Euclidean Geometry:** k=0.0 for theoretical physical modeling in a vacuum.
